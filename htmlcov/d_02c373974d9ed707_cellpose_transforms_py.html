215def normalize_percentile(Y: np.ndarray, lower: float = 1, upper: float = 99): 

216 """normalize image so 0.0 is lower percentile and 1.0 is upper percentile 

217 Percentiles are passed as floats (must be between 0 and 100) 

218 

219 Args: 

220 Y: The image to be normalized 

221 lower: Lower percentile 

222 upper: Upper percentile 

223 

224 """ 

225 X = Y.copy() 

226 x01 = np.percentile(X, lower) 

227 x99 = np.percentile(X, upper) 

228 X = (X - x01) / (x99 - x01) 

229 return X
